1. 议程设置理论\n**考试频率**：极高\n**核心要点**：\n- 提出者：McCombs & Shaw（1972）\n- 核心观点：媒介不能决定\"怎么想\"，但能决定\"想什么\"\n- 两个层面：第一层（议题设置）+ 第二层（属性设置）\n- 新媒体时代的演变：网络议程设置\n\n**答题技巧**：看到\"媒介告诉人们想什么\"选议程设置\n\n### 2. 3. 使用与满足理论\n**考试频率**：中等\n**核心要点**：\n- 范式转变：从\"媒介对人们做了什么\"到\"人们用媒介做了什么\"\n- 受众主动性：选择性、目的性\n- 五种需求：认知、情感、个人整合、社会整合、舒解压力\n- 局限性：过于强调理性，忽视媒介的潜移默化影响\n\n### 4. 两级传播理论\n**考试频率**：高\n**核心要点**：\n- 第三人效果\n**考试频率**：高\n**核心要点**：\n- 提出者：Davison（1983）\n- 两层含义：感知假设（对他人影响大）+ 行为假设（导致行动）\n- 产生原因：自我强化、低估他人、自我服务偏差\n- 应用领域：审查制度、广告效果、负面信息传播\n\n**答题技巧**：看到\"认为媒介对别人影响更大\"、\"支持审查/限制\"等关键词\n\n### 9. 框架理论\n**考试频率**：极高\n**核心要点**：\n- 来源：Goffman（1974）\n- 核心观点：框架是人们认识和解释世界的认知结构\n- 媒介框架：选择、强调、排除\n- 与议程设置的区别：想什么 vs 怎么想\n\n**重要考点**：新闻框架的四个功能（问题界定、因果解释、道德评价、解决方案）\n\n### 10. 把关人理论\n**考试频率**：较低\n**核心要点**：\n- 概念来源：Lewin（1947，心理学）→ White（1950，新闻学）\n- 把关标准：新闻价值、媒体立场、组织规范\n- 新媒体时代的演变：算法把关、用户把关\n\n---\n\n## 记忆技巧\n\n### 联想记忆法\n\n**议程设置** →
